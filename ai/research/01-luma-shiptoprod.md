## https://docs.guild.ai/ (Guild AI platform) — 2026-04-24

### What Guild Is
**"Control plane for AI agents"** — unified platform to build, deploy, govern AI agents.

### Architecture (3 layers)
1. **guildcore** — API layer (auth + request routing)
2. **Runtime** — agent executor (lifecycle, code execution)
3. **Agent Layer** — tools, LLM calls, sub-agent invocations

### Core capabilities
- Install agents from an **Agent Hub**
- Build custom agents via `@guildai/agents-sdk`
- Manage sessions (execution logs, input/output)
- Credential policies + context management
- Tool integration + LLM invocations
- 3 agent types: **auto-managed state**, **LLM-driven**, **self-managed state**
- Babel compiler plugin for TypeScript agent compilation
- Pre-built tool sets + integrations: GitHub, Jira, Slack, Azure DevOps, Bitbucket, Figma, Confluence
